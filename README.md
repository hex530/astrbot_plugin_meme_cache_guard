# AstrBot 表情包缓存卫兵 (Meme Cache Guard)

本插件为 AstrBot 提供了一个高效的表情包管理与省钱方案。通过缓存表情包的文字描述，避免重复调用昂贵的图片识别接口。

## 核心特性

- **过目不忘**：识别过的表情包将永久存入本地缓存，下次出现时秒懂。
- **大幅省钱**：通过“搬运”缓存描述，减少 90% 以上的图片识别 Token 消耗。
- **智能容量管理**：支持自定义缓存上限（默认 500 张），采用 FIFO 算法自动代谢旧记忆。
- **全格式支持**：支持 QQ 原生表情、商城表情以及普通图片。

## 安装方法

1. 在 AstrBot 插件管理中搜索 `meme_cache_guard` 或通过本仓库 URL 安装。
2. 重启 AstrBot。

## 配置项说明

| 配置项 | 说明 | 默认值 |
| :--- | :--- | :--- |
| `cache_limit` | 缓存的最大图片数量。 | `500` |
| `auto_recognize` | 是否开启自动识别并存入缓存。 | `true` |
| `admin_qq` | 管理员 QQ 号。 | `1591793025` |
